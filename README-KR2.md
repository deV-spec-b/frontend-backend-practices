## Практика 11: RBAC (ролевая модель)

### Реализовано:

#### Роли:
 Роль  Права 

 **Гость** (не авторизован) | Только регистрация и вход 
 **Пользователь** | Просмотр товаров, просмотр профиля 
 **Продавец** | Просмотр товаров + создание и редактирование товаров 
 **Администратор** | Все права продавца + управление пользователями 

#### Маршруты с учётом ролей:

 Маршрут  Метод  Доступ  Описание 
 `/api/auth/register`  POST  Гость  Регистрация 
 `/api/auth/login`  POST  Гость  Вход 
 `/api/auth/refresh`  POST  Гость  Обновление токенов 
 `/api/auth/me`  GET  Пользователь+  Данные текущего пользователя 
 `/api/users`  GET  Администратор  Список всех пользователей 
 `/api/users/:id`  GET  Администратор  Получить пользователя по id 
 `/api/users/:id`  PUT  Администратор  Обновить информацию пользователя 
 `/api/users/:id`  DELETE  Администратор  Заблокировать пользователя 
 `/api/products`  POST  Продавец+  Создать товар 
 `/api/products`  GET  Пользователь+  Получить список товаров 
 `/api/products/:id`  GET  Пользователь+  Получить товар по id 
 `/api/products/:id`  PUT  Продавец+  Обновить товар 
 `/api/products/:id`  DELETE  Администратор  Удалить товар 

#### Фронтенд для администратора:
